# 📘 Overview

**learnWebDev** is a comprehensive laboratory manual designed for university web development courses. It provides structured, hands-on labs covering the complete web development stack from frontend to backend and CMS deployment.

The curriculum follows industry-standard progression:
- **Frontend Fundamentals** → HTML/CSS layouts and forms
- **Responsive Design** → Bootstrap professional templates
- **JavaScript Interactivity** → Dynamic page enhancements
- **Backend Development** → PHP/MySQL dynamic websites
- **Frameworks & CMS** → CodeIgniter MVC + WordPress deployment

---

# 🎯 Lab Objectives

This laboratory manual helps students master:
- Build responsive websites using modern HTML5/CSS3/Bootstrap
- Create interactive web applications with JavaScript/jQuery
- Develop dynamic database-driven websites with PHP/MySQL
- Implement MVC architecture using CodeIgniter framework
- Customize and deploy WordPress themes/plugins
- Follow professional web development workflows and best practices

---

# 🛠️ System Requirements

## Essential Tools
- **IDE:** VS Code / Sublime Text
- **Browser:** Chrome / Firefox (DevTools)
- **Server:** XAMPP (Apache + MySQL + PHP)
- **Tools:** Git, Bootstrap CDN, jQuery CDN

## Lab-Specific Requirements

| Lab Group | Requirements |
|-----------|--------------|
| Frontend (1-9) | VS Code + Browser |
| Backend (10-12) | XAMPP + MySQL |
| Frameworks (13-14) | XAMPP + Composer |

# 📂 Lab Index

## Frontend Fundamentals
- **Lab 01: IDE & Basic HTML**  
  *Objective: Setup IDE and create first HTML page*
- **Lab 02: HTML Tables, Forms & Media**  
  *Objective: Design structured content with tables/forms*
- **Lab 03: HTML Iframes**  
  *Objective: Embed external content*
- **Lab 04: CSS Text & Layout**  
  *Objective: Style text and create page layouts*

## Responsive Design
- **Lab 05: Bootstrap Layout**  
  *Objective: Build responsive layouts*
- **Lab 06: Professional Bootstrap Pages**  
  *Objective: Create responsive professional websites*
- **Lab 07: Custom Bootstrap Templates**  
  *Objective: Modify and customize templates*

## JavaScript & Interactivity
- **Lab 08: JavaScript Basics**  
  *Objective: Add interactivity to web pages*
- **Lab 09: jQuery Effects**  
  *Objective: Enhance pages with jQuery animations*

## Backend Development
- **Lab 10: XAMPP & MySQL**  
  *Objective: Setup server and create databases*
- **Lab 11: PHP CRUD Operations**  
  *Objective: Build Create/Read/Update/Delete*
- **Lab 12: Dynamic PHP Website**  
  *Objective: Complete dynamic website*

## Frameworks & CMS
- **Lab 13: CodeIgniter Framework**  
  *Objective: MVC development with CodeIgniter*
- **Lab 14: WordPress Development**  
  *Objective: Theme customization & plugins*

# ▶️ How to Run Labs

## Frontend Labs (01-09)
```bash
# Clone repository
git clone https://github.com/ft-FiasCode/learnWebDev.git

# Open any lab folder in VS Code
cd learnWebDev/lab01-ide-html/
code .

# Open index.html in browser
Backend Labs (10-14)
bash
# 1. Install and start XAMPP (Apache + MySQL)
# 2. Copy lab folder to htdocs/learnWebDev/lab10/
