# 📘 Overview

**learnWebDev** is a comprehensive laboratory manual designed for **university web development courses**.
It provides structured, hands-on labs covering the **complete web development stack** from frontend to backend and CMS deployment.

### Curriculum Progression

* **Frontend Fundamentals** → HTML/CSS layouts and forms
* **Responsive Design** → Bootstrap professional templates
* **JavaScript Interactivity** → Dynamic page enhancements
* **Backend Development** → PHP/MySQL dynamic websites
* **Frameworks & CMS** → CodeIgniter MVC + WordPress deployment

---

# 🎯 Lab Objectives

This laboratory manual helps students master:

* Build responsive websites using **HTML5 / CSS3 / Bootstrap**
* Create interactive web applications with **JavaScript / jQuery**
* Develop dynamic database-driven websites with **PHP / MySQL**
* Implement **MVC architecture using CodeIgniter**
* Customize and deploy **WordPress themes and plugins**

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ft-FiasCode/learnWebDev.git
cd learnWebDev
```

Install and start the required environment:

* Install **XAMPP**
* Start **Apache** and **MySQL**
* Place project folder inside `htdocs`

Example path:

```
xampp/htdocs/learnWebDev
```

---

# ▶️ Usage

## Frontend Labs (01–09)

```bash
cd learnWebDev/lab01-ide-html
code .
```

Open **index.html** in your browser.

---

## Backend Labs (10–14)

1. Start **Apache and MySQL** in XAMPP
2. Copy lab folder into `htdocs`
3. Import **database.sql** into phpMyAdmin
4. Open in browser:

```
http://localhost/learnWebDev/lab10
```
